# colors-lamp
Repository for COLORS lab using LAMP stack

The COLORS application is a full-stack web application built using LAMP stack
It allows user to 
- login/register
- Add colors to database
- Search database for colors

Technologies Used
- Ubuntu
- Apache
- MySQL
- PHP
- html
- css
- javascript
- Postman (for API testing)

Setup Instructions:
1) Get a remote Linux server running LAMP stack on Digital Ocean
2) Create MySQL database
3) Implement backend logic
4) Add html, css, and javascript files for frontend
